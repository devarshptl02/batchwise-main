### 🌟 Project Overview

Managing a tuition center involves juggling student records, tracking daily attendance, and monitoring fee payments. 

**This project provides a centralized dashboard to handle all these administrative bottlenecks, allowing educators to focus entirely on teaching.** ---

### ✨ Key Features

- **👨‍🎓 Student Management:** Easily enroll new students, update their profiles, and maintain organized batches.

- **📅 Smart Attendance:** Track daily attendance with a simple interface and generate quick reports.

- **💰 Fee Tracking:** Keep a clear ledger of pending and completed fee payments to avoid manual errors.

- **📊 Dashboard Analytics:** Get a quick overview of total students, active batches, and monthly revenue.

---

### 🛠️ Technology Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Node.js / Express 
- **Database:** Supabase 
- This project is vibe-coded, with supporting us for more better practical learning.

### 🚀 Local Setup Instructions

To run this project locally, simply download or clone the repository to your computer. 

Open the project folder in your preferred code editor, and then launch the main HTML file directly in any web browser to view and interact with the application.
